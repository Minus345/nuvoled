#Nuvoled
Nuvoled Application in Java  
 - **Java 17.0.1 require**
 - 
***
##Start Parameters
###Test Connection
`java -jar nuvoled.jar start [ip]`  
[ip] = broadcast ip
###Send Color
`java -jar nuvoled.jar start [ip] color [R] [G] [B] `  
[R] [G] [B] = 0 - 255
###Send Picture
`java -jar nuvoled.jar start [ip] picture`
###Show Screen
`java -jar nuvoled.jar start [ip] screen [x] [y]`  
[x] [y] = pixel
